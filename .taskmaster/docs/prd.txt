# AI GUIDE TOUR - Product Requirements Document

## 1. 프로젝트 개요

### 1.1 프로젝트명
**AI GUIDE TOUR**

### 1.2 프로젝트 비전
AI 기술을 활용하여 개인 맞춤형 관광 가이드를 제공하는 **모바일 우선** 플랫폼

### 1.3 핵심 차별화 포인트
1. **AI 기반 완전 개인화**: 사용자의 관심사, 시간, 배경지식에 맞춘 실시간 콘텐츠 생성
2. **극도로 간단한 UX**: 검색 → 선택 → 투어 시작 (3단계 완료)
3. **모바일 퍼스트**: 한 손으로 쉽게 조작 가능한 직관적 인터페이스

## 2. 핵심 기능 명세

### 2.1 MVP (Minimum Viable Product) 기능

#### 2.1.1 검색 및 발견
- **통합 검색**: 관광명소, 유적지, 박물관 등 통합 검색
- **AI 기반 추천**: 사용자 위치 기반 주변 명소 추천
- **카테고리 필터**: 역사, 문화, 자연, 종교 등 카테고리별 분류

#### 2.1.2 가이드 콘텐츠 생성
- **AI 개요 생성**: 명소의 역사적 배경, 문화적 의미, 건립 배경
- **관람 동선 추천**: 효율적이고 논리적인 관람 순서 제안
- **포인트별 설명**: 각 구간별 상세 설명 및 흥미로운 에피소드

#### 2.1.3 멀티미디어 가이드
- **텍스트 가이드**: 상세한 설명과 배경 정보
- **오디오 가이드**: TTS 기반 음성 안내 (걷는 속도에 맞춘 타이밍)
- **이미지 갤러리**: 관련 사진 및 복원도

#### 2.1.4 개인화 기능
- **관심사 설정**: 역사, 건축, 예술 등 개인 관심 분야
- **투어 시간 설정**: 30분, 1시간, 2시간 등 시간대별 동선
- **난이도 조절**: 일반인, 전문가 수준의 설명 깊이

## 3. 기술 스택

### 3.1 Frontend
- **Framework**: Next.js 14 (App Router) + PWA
- **UI Library**: React 18
- **Styling**: TailwindCSS + Radix UI
- **State Management**: Zustand
- **HTTP Client**: TanStack Query

### 3.2 Backend
- **Runtime**: Node.js 20+
- **Framework**: Next.js API Routes
- **Database**: PostgreSQL + Prisma ORM
- **Cache**: Redis
- **AI Integration**: OpenAI GPT-4, Anthropic Claude

### 3.3 AI 시스템
- **Main AI**: OpenAI GPT-4 Turbo (콘텐츠 생성)
- **TTS**: ElevenLabs (프리미엄), Google TTS (기본)
- **개인화**: 실시간 사용자 행동 분석

## 4. 개발 로드맵

### Phase 1: MVP 개발 (10주)
1. **Week 1-2**: AI 엔진 & 모바일 기반 구축
2. **Week 3-4**: 핵심 AI 콘텐츠 생성
3. **Week 5-6**: 모바일 UX 최적화
4. **Week 7-8**: 수익화 시스템 구현
5. **Week 9-10**: 테스트 & 최적화

### Phase 2: 확장 & 개선 (8주)
1. **Week 11-14**: 콘텐츠 확장 & AI 고도화
2. **Week 15-18**: 고급 기능 & B2B

### Phase 3: 글로벌 확장 (12주)
1. **Week 19-26**: 다국어 & 글로벌화
2. **Week 27-30**: 플랫폼 고도화

## 5. 성공 지표 (KPI)

### 5.1 사용자 관련 지표
- **MAU**: 월간 활성 사용자 수
- **사용자 유지율**: 1일, 7일, 30일 유지율
- **세션당 사용 시간**: 평균 앱 사용 시간
- **투어 완료율**: 시작한 투어를 끝까지 완료한 비율

### 5.2 비즈니스 지표
- **프리미엄 전환율**: 무료 → 유료 전환 비율
- **ARPU**: 사용자당 평균 수익
- **CAC**: 사용자 획득 비용
- **LTV**: 사용자 생애 가치

## 6. 수익화 모델

### 6.1 메인 수익 모델
- **텍스트 가이드**: 100% 무료
- **기본 오디오**: 광고 시청 후 무료 (일일 3회 제한)
- **프리미엄 오디오**: 월 구독 9,900원 또는 개별 구매 1,500원

### 6.2 부가 수익 모델
- **광고 수익**: 동영상 광고, 네이티브 광고
- **B2B 파트너십**: 지자체, 호텔, 교육기관
- **데이터 수익화**: 관광 트렌드 분석

## 7. 기술적 요구사항

### 7.1 성능 요구사항
- **로딩 시간**: 첫 화면 3초 이내
- **AI 응답**: 가이드 생성 10초 이내
- **오디오**: 끊김없는 재생
- **오프라인**: 기본 기능 사용 가능

### 7.2 품질 요구사항
- **Lighthouse**: 90점 이상
- **접근성**: WCAG 2.1 AA 준수
- **보안**: API 인증, 데이터 암호화

## 9. 리스크 및 대응방안

### 9.1 기술적 리스크
- **AI 콘텐츠 품질**: 전문가 검수 시스템, 사용자 피드백 반영
- **서버 부하**: 클라우드 auto-scaling, CDN 활용

### 9.2 비즈니스 리스크
- **저작권 이슈**: 오픈소스 데이터 활용, 자체 콘텐츠 제작
- **경쟁사 출현**: 지속적인 혁신, 브랜드 차별화

## 10. 최종 목표

### 10.1 단기 목표 (1년)
- 월 사용자 5만 명
- 월 수익 800만원
- 서울/부산/경주 커버

### 10.2 장기 목표 (3년)
- 월 사용자 100만 명
- 월 수익 10억원
- 전 세계 1,000개 명소 커버
